# Restaurant CMS

A modern full-stack restaurant management platform designed to simplify restaurant operations through a dedicated admin dashboard and a customer-facing website.

## Project Structure

```
restaurant-cms/
├── server/         # Backend API
├── cms-client/     # Admin Dashboard
└── web-client/     # Public Website
```

### Features

* Authentication & Authorization
* Menu Management
* Category Management
* User (Non Customer) Management
* Image Upload
* Role-based Access Control
* Responsive Admin Dashboard
* Public Restaurant Website
* RESTful API
* Database Integration

Tech Stack

Backend

* JavaScript
* Express
* PostgreSQL
* Sequelize ORM
* JWT Authentication
